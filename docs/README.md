# Documentation

This directory contains all project documentation organized by purpose.

## Structure

| Directory | Purpose | Files |
|-----------|---------|-------|
| `guides/` | How-to and conventions | CONVENTIONS.md, I18N_SETUP.md |
| `reference/` | Architecture, vision, strategy | VISION.md, PROJECT_CONTEXT.md |
| `reports/` | Audits, test reports, analysis | ANALYSIS_REPORT.md, audit-report.md, diagnostic-workout-plan.md, e2e-test-report.md, onboarding-e2e-report.md, rnd-report.md |
| `maintenance/` | Logs and maintenance records | (empty) |
| `agents/` | Cascade agent configuration | AGENTS.md |
| `domains/` | Domain-specific documentation | core/, planning/, storage/, ui/ |
| `screenshots/` | UI screenshots for docs | *.png |
